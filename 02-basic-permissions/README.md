# Basic Permissions

Begin by changing to the practice directory:

```
cd ~/linux-practice
```

Let's go ahead and list out the file structure and see what we have so far:

```
ls -l ~/linux-practice
```

These lessons will be used to see how changing the permissions can change the behavior of the activity you might try to do but also test if you know what bits do what.

#### Assignment #1
Change directory to ***dir1***

Let's change the permission of ***one.txt***. Please set the permission to only the world can ***read*** it

If you try to cat that file, what happens?

Let's change the permission of ***two.txt***. Please set the permission to where only the world can ***execute*** it

If you try to cat that file, what happens?

Let's change the permission of ***three.txt***. Please set the permissions to read and write for both owner and group. Then give no permissions to world

If you try to cat that file, what happens?
